MAMBO EVENTS COMPONENT

April 2006 :: Version 1.3 beta

ATTENTION

Please open
Components :: Events :: Edit config
and change the Admin email address!

Then, open
Components :: Events :: Manage events categories
and set up your categories. You have to set up and publish at least one before you can add any event.

Upgraders: If you are upgrading please save your configuration again as new default values were added.

HISTORY

Version 1.3:
  • enabling of caching from component and modules - major performance benefits
  • full Joomfish compatability
  • Slovenian translation added, Spanish improved
  • Finnish, Danis and Greek translations added
  • allow all users by config option to add events (still must be approved)
  • removed need for adding an activity
  • allow publishing of events from frontend
  • many small bug fixes
